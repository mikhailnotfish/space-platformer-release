# what i did for eight hours 😭
i started this project, almost a week ago and have been using every last second away from school i have, to make it.
there were a LOT of setbacks, which is partly why this project took so long. one time, my computer had an update, and i lost like 2 hours of work because i didnt save anything!
altogether, the actual coding didnt take that long. c# is a pretty simple language. what took me so long, was making the map. every time i added a new section, i had to go, and beat the whole game
to be able to see if that section was beatable or not! then, i needed to duplicate an infinite amount of cubes, and scale them to fitting size. this took an incredable amount of time.
